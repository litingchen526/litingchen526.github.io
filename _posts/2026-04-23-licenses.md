### Plot 1: Top License Types

<div id="vis1"></div>

This chart shows the ten most common license types in the dataset. I used a horizontal bar chart because it allows for easy comparison across categories, especially since some license type names are long. In this chart, the x-axis encodes the number of licenses (a quantitative variable), while the y-axis encodes the license type (a categorical variable). I sorted the bars in descending order so that the most common license types appear at the top, making the chart easier to interpret at a glance.

Color is used to distinguish between different license types. Since license type is a nominal variable with no inherent ordering, a categorical color scheme is appropriate. The use of distinct colors helps separate the categories visually and improves readability without implying any ranking beyond what is already shown by position.

For data transformations, I grouped the dataset by license type and counted the number of records in each category. I then selected only the top ten license types to reduce clutter and avoid overwhelming the viewer with too many categories. This filtering step is important because the full dataset contains many less frequent license types that would make the chart harder to read.

### Plot 2: License Trends Over Time

<div id="vis2"></div>

This visualization shows how the number of issued licenses changes over time for selected license types. I used a line chart because it is effective for displaying trends across time. The x-axis represents the issue year (an ordinal/temporal variable), and the y-axis represents the number of licenses issued in each year (a quantitative variable). Points are added on the line to make individual yearly values easier to identify.

Color is used to represent license type, which allows different categories to be compared within the same visualization. However, instead of displaying all categories at once, the chart is designed to show one selected license type at a time. This avoids visual clutter and makes it easier to focus on a single trend.

To prepare the data, I first converted the “Original Issue Date” column into a datetime format and extracted the year into a new variable. Then, I grouped the data by issue year and license type and calculated the count of licenses for each combination. I also filtered the data to include only recent years (2000–2022), which removes extreme historical values and makes the trend more interpretable.

From the chart, we can observe that the default license type which is "DETECTIVE BOARD" shows fluctuations over time, with some peaks around the mid-2010s and a noticeable decline in recent years. This kind of pattern would be difficult to see clearly without aggregating and filtering the data.

### Interactivity

The second chart includes an interactive dropdown menu that allows the user to select a specific license type. When a selection is made, the chart updates to display only the trend for that category. This interaction is implemented using a selection filter, which dynamically controls which data is shown.
